# PoroAuth (魄罗通行证)

基于硬件级底层设备模拟（DD 驱动）的 WeGame 自动化辅助工具。化繁为简，解放双手 🚀。

## 1.1 项目简介
由于英雄联盟特定账号（如金币号、头盔号）的登录过程高度绑定 Tencent WeGame 客户端，且该终端频繁存在无法持久保存密码、异地环境校验繁杂等问题，导致每次启动游戏都极度耗时。
本项目致力于解决这一痛点，通过全本地的 AES-256 数据防固结合物理层级的键鼠驱动模拟，实现账号信息的可视化管理与无视防篡改检测的安全自动注入。

## 1.2 使用说明
- **环境要求**：为实现底层键鼠外设握手，启动与构建时必须具备相应的**管理员权限**才能完成 DLL 注册与映射。
- **核心依赖**：出于风控安全考虑，本仓库自身**不包含**核心注入模块。初次打开后界面将启用拦截保护，请依照指示前往 [ddxoft/master](https://github.com/ddxoft/master) 自行查阅并绑定合规的 `dd63330.dll` 引擎文件。

## 1.3 特性一览
- **通行名册**：可视化专属账号卡片网格，数据本地强加密。
- **智能录入**：支持直接从网购的发货平台复制 `账号----密码----附加信息`，在新增账号时智能识别切分。
- **动态隔离区**：防手病态误触支持！可对账号独立设置长达数天的封禁/冻结锁定倒计时。锁定期间相关卡片全部灰调、按钮物理失效，阻绝进一步刺激安全中心。
- **物理通信**：模拟纯鼠标轨迹与原生设备击键反馈，绝非简单的句柄发信。

本文介绍如何从零搭建及自购构建 PoroAuth：

## 3.1 运行平台
- Node.js 及 npm。
- Windows 10/11 x64（必须兼容底层系统 API 控制）。

## 3.2 Electron 主程序

```bash
# 获取并安装组件包
npm install

# 拉起高权限开发环境终端（含代码热重载 HMR）
npm run dev:admin

# 构建产出为 Windows 系统原生的绿色免安装/安装版本 .exe
npm run build:win
```
